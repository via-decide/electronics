# Glossary

| Term | Meaning |
| --- | --- |
| ADC | Analog-to-digital converter. Converts analog voltage into a digital code. |
| DMA | Direct memory access. Hardware-assisted data movement that avoids CPU copy loops. |
| ISR | Interrupt service routine. A short handler executed in interrupt context. |
| RTOS | Real-time operating system. Schedules tasks with priorities, queues, timers, and synchronization. |
| Jitter | Variation in event timing relative to the ideal schedule. |
| Latency | Time between cause and observable effect. |
| Throughput | Data processed per unit time. |
| Watchdog | Timer that resets or reports a fault if supervised code stops making progress. |
| Brownout | Supply voltage drop below the safe operating threshold. |
| OTA | Over-the-air firmware update. |
| NVS | Non-volatile storage used by ESP-IDF for persistent key-value data. |
| EMC | Electromagnetic compatibility. The ability to operate without causing or suffering unacceptable interference. |
| EMI | Electromagnetic interference. Unwanted coupling or emissions that disturb operation. |
| HALT/HASS | Highly accelerated life/stress screening methods for reliability discovery and production screening. |
